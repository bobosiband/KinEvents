Design a complete modern UI/UX system in Figma for **KinEvents** — a private family event and birthday management platform.

Repository context:

* Project name: **KinEvents**
* Purpose: **Family Events Management System** 
* The platform is designed for a small private family network (~20 users)
* The experience should feel like a secure digital family home

The entire design system must be:

* Mobile-first
* Responsive for desktop/tablets
* Warm and personal
* Modern and polished
* Extremely easy to use for all ages
* Built with reusable UI components
* Production-ready for frontend implementation

---

# Overall Product Feel

The UI should feel like:

* A private family community
* A shared family calendar + memory hub
* Warm, safe, emotional, and welcoming
* Simple enough for non-technical users

Avoid:

* Corporate SaaS aesthetics
* Business analytics dashboards
* Cold enterprise styling
* Public social media vibes

---
