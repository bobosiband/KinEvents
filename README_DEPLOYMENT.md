## What Changed

### Problem
```
Lambda A: in-memory cache = event "Original"
Lambda B: in-memory cache = event "Original" (stale from different cold-start)

Admin updates on A → title = "Updated"
User queries on B → returns "Original" ❌ (stale!)
```

### Solution
```
Every request:
1. await readData() → hits MongoDB
2. In-memory cache = fresh data from DB
3. Process mutations
4. await persistData() → writes to MongoDB
5. Both instances always see same state ✅
```
